# 🧠 Memory Game

A simple browser-based memory game where players must remember a number shown on the screen and enter it after it disappears. The game becomes progressively harder as the number of digits increases with each level.

## Features

* Increasing difficulty with each level
* Score and level tracking
* Instant feedback on wrong answers
* Simple and interactive UI
* Built using pure HTML, CSS, and JavaScript

## How to Play

1. Click **Start the Game**.
2. A random number will appear on the screen.
3. Memorize the number before it disappears.
4. Enter the number correctly to move to the next level.
5. If you enter the wrong number, the game ends and your score is shown.

## Tech Stack

* HTML
* CSS
* JavaScript

Memory Game

Instructions on How to play the Game:

  1. A Random number appears on the screen, After few seconds the Number disappears.


  2. you have to remember the Number before it Disappears and Enter the number in the Input Box.


  3. If You have entered Correct number Then the Level gets harder by increasing the digits of the Number.


  4. If you have entered wrong number then the Game ends & the number, your Level, Score, appears on the screen.


Tips: Activate the Num Lock on the right side of keyboard and start playing the game for better Game Experience
